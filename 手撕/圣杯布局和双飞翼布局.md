### 圣杯布局和双飞翼布局

### 圣杯布局

```html
<html>
    <style>
        *{
            margin: 0%;
            padding: 0%;
        }
        .header{
            height: 100px;
            background-color: green;
        }
        .content{
            display: flex;
            background-color: gray;
            bottom: 100px;
            height: calc(100vh - 200px);
        }
        .footer{
            position: absolute;
            height: 100px;
            width: 100%;
            bottom: 0%;
            background-color: green;
        }
        .left,.middle,.right{
            height: 100%;
        }
        .left,.right{
            background-color: orange;
            flex-basis: 30%;
        }
        .left{
            order:1;
        }
        .right{
            order: 3;
        }
        .middle{
            background-color: bisque;
            flex-basis: 40%;
            order: 2;
        }
    </style>
    <body>
        <div class="header">

        </div>
        <div class="content">
            <div class="middle">

            </div>
            <div class="left">

            </div>
            <div class="right">

            </div>
        </div>
        <div class="footer">

        </div>
    </body>
</html>
```

### 双飞翼布局

```html
<html>
    <style>
        *{
            margin: 0%;
            padding: 0%;
        }
        .header{
            height: 100px;
            background-color: green;
        }
        .content{
            background-color: gray;
            bottom: 100px;
            height: calc(100vh - 200px);
            overflow: hidden;
        }
        .footer{
            position: absolute;
            height: 100px;
            width: 100%;
            bottom: 0%;
            background-color: green;
        }
        .left,.middle,.right{
            height: 100%;
        }
        .left,.right{
            width: 30%;
            background-color: orange;
        }
        .left{
            position: relative;
            top: calc(200px - 100vh);
            width: 30%;
            float: left;
        }
        .right{
            top: calc(200px - 100vh);
            position: relative;
            margin-left: 40%;
            float: left;
        }
        .middle{
            float: left;
            background-color: bisque;
            width: 100%;
        }
    </style>
    <body>
        <div class="header">

        </div>
        <div class="content">
            <div class="middle">

            </div>
            <div class="left">

            </div>
            <div class="right">

            </div>
        </div>
        <div class="footer">

        </div>
    </body>
</html>
```
